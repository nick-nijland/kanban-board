# KanbanBoard

This project was created as an assessment for **PinkRoccade GGZ**.  
It is a Kanban board application built with **Angular**, using **Angular Material** for UI components and **Mock Service Worker (MSW)** for API mocking during development.

## Scripts

You can run the following commands using **npm**:

```bash
# Angular CLI
npm run ng

# Serve the application locally (uses `ng serve`)
npm run start

# Build the project
npm run build

# Watch for changes and rebuild automatically (development mode)
npm run watch

# Run tests
npm run test

# Format code using Prettier
npm run prettier
